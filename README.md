# Global Fertilizer Trade Analysis Dashboard

## Project Overview

This project presents an interactive Excel dashboard for analyzing global fertilizer trade data. The dashboard provides insights into fertilizer import and export trends, country-wise trade performance, compound distribution, and year-wise trade value using Microsoft Excel's analytical and visualization features.

The primary objective of this project is to transform raw trade data into meaningful visualizations that support data-driven decision-making.

---

# Objectives

* Analyze global fertilizer trade data.
* Identify top countries based on trade value.
* Study year-wise trade trends.
* Examine the distribution of fertilizer compounds.
* Develop an interactive dashboard using PivotTables and PivotCharts.

---

* Element

## Data Preprocessing

The following preprocessing steps were performed:

* Removed unnecessary columns such as Area Code, Item Code, Element Code, Unit, and Flag.
* Renamed columns for improved readability.
* Standardized the dataset structure.
* Verified data consistency and removed redundant information.
* Converted the dataset into an Excel Table for efficient analysis.

## Final Dataset

The processed dataset contains the following columns:

* Country
* Compounds
* Element
* Year
* Revenue

# Dashboard Features

The dashboard includes the following analytical components:

### Year-wise Revenue Analysis

Displays the trend of fertilizer trade value across different years.

### Top Country Revenue

Shows countries contributing the highest trade value.

### Elements Distribution

Displays the distribution of trade elements including:

* Agricultural Use
* Production
* Import Quantity
* Import Value
* Export Quantity
* Export Value

### Top Ten Compounds

Shows the most frequently traded fertilizer compounds.

# Key Insights

* Fertilizer trade value generally increased between 2002 and 2017.
* A small number of countries account for a significant portion of global trade value.
* Import Quantity represents the largest share among the trade elements.
* Ammonium Nitrate, Urea, and Potassium Sulphate are among the most frequently traded fertilizer compounds.
